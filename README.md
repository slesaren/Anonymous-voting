# Система анонимного голосования со слепой подписью

Данный проект реализует протокол **слепой подписи** на основе алгоритма **RSA** для обеспечения анонимности избирателей в электронном голосовании.

## Принцип работы
Проект имитирует сценарий, при котором избиратель хочет отдать свой голос, но сервер **не должен знать, кто именно и за что проголосовал**. Это достигается в три шага:

1. **Ослепление (Blinding)**: Избиратель (Клиент) генерирует случайное число `r`, умножает на него свой бюллетень и отправляет эту "абракадабру" Серверу.
2. **Подпись вслепую**: Сервер, не видя содержимого бюллетеня, подписывает его своим секретным ключом.
3. **Снятие ослепления**: Клиент снимает соринку `r` с подписи и получает **действительную подпись** Сервера под своим настоящим бюллетенем. Затем он анонимно (или с минимальной мета-информацией) отправляет уже подписанный бюллетень на подсчет.


## Требования
- **Java Development Kit (JDK)** версии 8 или выше.
- **Visual Studio Code** (рекомендуется) с установленным расширением `Extension Pack for Java`.
- *Или любая другая IDE (IntelliJ IDEA, Eclipse).*

## Установка и запуск

### Способ 1: Запуск в VS Code
1. Убедитесь, что файлы лежат строго в пакете `org.example` (папка `src/main/java/org/example`).
2. Откройте корневую папку `голосование` в VS Code.
3. Нажмите `Ctrl+Shift+P` -> **`Java: Create Java Project`** -> **`No build tools`**. Выберите текущую папку.
4. Откройте файл **`VotingGUI.java`**.
5. Нажмите кнопку **`Run`** над методом `main`.

### Способ 2: Запуск через терминал
Скомпилируйте и запустите вручную из корня проекта:
```bash
# Переход в папку с исходниками
cd src/main/java

# Компиляция
javac org/example/*.java

# Запуск
java org.example.VotingGUI

```
## Как использовать
- После запуска откроется окно "Система анонимного голосования". Выберите Избирателя (Иванов, Петров и т.д.) и вариант Голоса (Да/Нет/Воздержался).

- Нажмите "1. Создать бюллетень". Клиент сгенерирует "слепой" бюллетень.

- Нажмите "2. Получить слепую подпись". Сервер подпишет его, не зная содержимого.

- Нажмите "3. Отправить голос". Клиент "снимет ослепление" и отправит готовый голос.

- Нажмите "Показать результаты", чтобы увидеть текущий подсчет.

- Чтобы проголосовать за другого избирателя, нажмите "Сброс для нового избирателя".
